## Install and Setup HelloWorld in Vue.js  
These instructions will create an app in a `vue-app` folder.

**Install the Vue CLI version 3**  
`npm i -g @vue/cli` 

**Using the CLI create an app and give it a name**  
`vue create vue-cl-test`  

**Run the app locally**
```
cd vue-cl-test
npm run serve
```
